Generate Your Personal MCP Endpoint

Your MCP endpoint is the secure connection point that allows your AI assistant to access AI-MCP-Bio's vast Bioinformatic resources. Think of it as a secure API key that grants your AI controlled access to specialized tools.

Getting Your MCP URL

To generate your personal MCP endpoint:

  1. Log in to your AI-MCP-Bio account
  2. Navigate to the MCP Settings page
  3. Click on Generate New Endpoint
  4. Copy your unique MCP server URL

Caution: Treat your MCP server URL like a password! It can be used to run your AI-MCP-Bio tools and access your data.

Configuring Cursor IDE

For Cursor version ‚â• 0.46 (recommended):

  1. Open Cursor settings (‚áß+‚åò+J)
  2. Navigate to the "MCP" tab and click "Add MCP Server"
  3. Add the following JSON configuration, replacing the placeholder URL with your MCP URL from AI-MCP-Bio:
{
  "mcpServers": {
    "AI-MCP-Bio": {
      "url": "https://mcp.ai-mcp-bio.org/endpoint/YOUR_MCP_KEY/sse"
    }
  }
}

Note: Replace YOUR_MCP_KEY with your actual MCP key from the settings page.

For Older Cursor Versions

In older versions of Cursor (< 0.46), you can set up AI-MCP-Bio by:

  1. Going to Settings > MCP
  2. Directly pasting your full MCP URL in the input field
  3. Clicking "Save" to complete the setup

Configure Your AI Assistant Capabilities

After generating your MCP endpoint, you need to configure which bioinformatics tools and resources your AI assistant can access. This gives you precise control over your AI's capabilities.

Available Tool Categories

AI-MCP-Bio offers a wide range of specialized bioinformatics tools:

  • Genomic Analysis Tools
    • BLAST - Search DNA and protein databases
    • BWA - Map DNA sequences against reference genomes
    • GATK - Variant discovery in high-throughput sequencing data
    • Samtools - Manipulate alignments in SAM/BAM format
  • Structural Biology Tools
    • AlphaFold - Protein structure prediction
    • PyMOL - Molecular visualization
    • Docking tools - Predict ligand binding
  • Pathway and Systems Biology
    • KEGG - Pathway mapping
    • Reactome - Pathway analysis
    • Gene Ontology tools - Functional annotation
  • Database Access
    • GenBank - Genetic sequence database
    • UniProt - Protein sequence and function
    • PDB - Protein structure database

Configuring Tool Access

To configure which tools your AI can access:

  1. Navigate to your AI-MCP-Bio dashboard
  2. Go to the "Configure Tools" section
  3. Enable or disable specific tools based on your research needs
  4. Set access levels and permissions for each tool
  5. Save your configuration

Pro Tip: Start with just the essential tools you need for your current research. You can always add more tools later as your needs expand.

Setting Usage Limits

For better resource management and cost control, you can set usage limits:

  • API call limits per day/week/month
  • Computation time allowances
  • Storage quotas for results
  • Maximum parallel jobs

Connect Your AI Assistant

With your MCP endpoint generated and tools configured, the final step is connecting your AI assistant to use these powerful capabilities.

Compatible AI Assistants

AI-MCP-Bio currently works with several AI systems:

  • Cursor IDE - Integrated development environment with AI coding capabilities
  • Claude by Anthropic - Via Claude Desktop application
  • Custom LLM Applications - Through our API

Using with Cursor IDE

Once you've configured your MCP server in Cursor settings (as described in Step 1), you can immediately start using AI-MCP-Bio capabilities:

  1. Open any project in Cursor
  2. Open the AI chat panel (Ctrl+J or Cmd+J)
  3. Ask your AI to perform bioinformatics tasks like:
    • "Analyze this FASTA sequence using BLAST"
    • "Run a variant calling pipeline on this genome data"
    • "Compare these protein structures and highlight the binding sites"
// Example conversation in Cursor

You: Can you help me analyze this FASTA sequence and identify any known mutations?

AI: I'd be happy to analyze this sequence for mutations. I'll use the BLAST tool 
through the MCP connection to identify the sequence and check for variants.

[AI is requesting to use the BLAST MCP tool]

You: [Approve tool use]

AI: I've analyzed your sequence. It's a variant of the TP53 tumor suppressor gene. 
I found 3 notable mutations:
1. R175H - Known pathogenic mutation often found in various cancers
2. G245S - Less common variant with moderate impact on protein function
3. R282W - Affects DNA binding capability of the p53 protein

Would you like me to run additional analyses on these mutations?

Tool Permissions

When your AI assistant attempts to use a bioinformatics tool, you'll receive a permission prompt. You can:

  • Approve - Allow the tool to be used for this request
  • Deny - Prevent the tool from being used
  • Always approve - Enable automatic usage of this tool in the future

Note: For sensitive research data, we recommend reviewing each tool request before approval to maintain data security.
